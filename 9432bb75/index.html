中国农历是如何计算的

发表于 2023-04-07 分类于 读书笔记 阅读次数：

根据如下一些规则确定农历每一天是几月几日. 这些规则根据太阳运行的规律确定节气, 根据月亮运行的规律确定月份, 所以被叫做阴阳合历.

第一步, 定气

地球上的人观察星空发现太阳在天球上的一条轨道上做圆周运动, 这条轨道被叫做黄道. 把黄道圆周平均分成24分, 每份是15°. 把黄道和天球赤道的两个交点中的冬至点作为起始的节气冬至, 太阳每走15°就定一个节气.把从冬至点开始每30°的节气又叫做中气.

  1. 节气是即太阳运行到一定角度的那个时刻所在的那一天. 一天的概念后面会介绍.
  2. 太阳从前一个冬至点到下一个冬至点之间的时间叫做一个回归年, 也叫太阳年.
  3. 1回归年上中天≈365.2421990741日≈365天5小时48分46秒.
  4. 太阳在黄经上运行并不是匀速, 在近地点速度快, 而远地点速度慢, 所以节气之间的时间并不均匀.

第二步, 定月

太阳和月亮在天球上运动时处于相同黄经的时刻称为朔或合朔. 两次和合朔的时间间隔为一个朔望月. 朔望月长度不是定值, 大约在29.27到29.83天之间变动, 平均长度为29.530588天, 即29天12小时44分钟2.8秒.

每个合朔时刻所在的那一天为朔日, 定为每个月的初一. 之后每天就是初二, 初三... 一个月有可能是29天也有可能是30天. 直到下一个朔日. 冬至所在的那个月定为每年的十一月, 也叫冬月,或者子月(子丑寅卯的子).

第三步, 置闰

两个冬至日之间为1回归年, 每个回归年约有12.37个月. 365.24/29.53≈12.37. 所以两个子月之间(不包括两子月)可能有11个月, 也可能有12个月. 如果是后一种情况, 那么这12个月里面要经历除了冬至以外的11个中气, 必然至少有一个月没有中气. 由于中气间隔不均匀, 所以也有可能不止1个月没有中气. 那么第一个没有中气的月份定为闰月. 如果不计算闰月, 那么子月后面的那个月为十二月(腊月, 丑月), 正月(寅月), 二月(卯月)...复至十一月. 闰月的名字就跟着前一个月走, 叫闰几月. 比如如果子月之后的那个月是闰月, 就叫闰子月或者闰十一月. 正月初一就是元旦.

概念

前面用到的一些概念, 为了阅读流畅, 在下面说明.

一天

有两种方式定义一天, 就是太阳日和恒星日. 日历里使用的是太阳日.

太阳连续两次经过同一地点上中天的时间间隔, 叫做一个太阳日.

即太阳连续两次升到最高点的时间间隔.

这里又分为真太阳日和平太阳日, 真太阳日就是真实测定的, 但由于真太阳日每天长度都在不断变化, 最长和最短相差51秒. 所以又定义了一个平太阳的概念, 就是假设有一个假的太阳, 匀速沿着天赤道作周年视运动, 运动速度等于黄道上太阳周年视运动的平均值, 这个假想的太阳就是平太阳. 平太阳计算出的太阳日就是平太阳日

平太阳日就是一年里真太阳日的平均长度.

恒星日经线两次对向同一恒星的时间间隔，恒星日是以遥远的恒星为参考系，是地球自转360度的周期.

日历里面使用的是平太阳日作为一天的概念, 平太阳日是24小时, 那么恒星日大概是24小时56分钟4秒.

天球(Celestial sphere)

天球是在天文学和导航上想出的一个与地球同球心，并有相同的自转轴，半径无限大的球。天空中所有的物体都可以当成投影在天球上的物件。地球的赤道和地理极点投射到天球上，就是天球赤道和天极。天球上的天体投影的坐标有三种标注方式, 分别是地平坐标系, 赤道坐标系, 和黄道坐标系. 介绍如下.

地平坐标系

地平坐标系是一种天球坐标系,

  1. 观测者所在的地平面无限扩展与天球相交的大圆为地平圈. (由于天球的半径是任意长的，而地球的半径则相对很小，因此，观测者所在的点可以认为是与地心重合的，地平圈也可以看成是以地心为圆心的, 所以地平圈为天球上的大圆。)
  2. 从观测者所在的地点，作垂直于地平面的直线并无限延长，在地平面以上与天球相交的点，称为天顶；在地平面以下与天球相交的点，称为天底。
  3. 通过天顶和天底可以作无数个与地平圈相垂直的大圆，称为地平经圈；也可以作无数个与地平圈平行的小圆，称为地平纬圈。
  4. 通过天顶和北天极的地平经圈（当然也通过天底和南天极），与地平圈有两个交点；靠近天北极的地个点为北点，靠近天南极的那个点为南点。北点和南点分别把地平圈和地平经圈等分。根据面北背南、左西右东的原则，可以确定当地的东点和西点，即面向北点左90°为西点，右90°为东点。
  5. 在地平坐标系中，通过南点、北点的地平经圈称子午圈。子午圈被天顶、天底等分为两个180°的半圆。以北点为中点的半个圆弧，称为子圈，以南点为中点的半个圆弧，称为午圈。
  6. 中天是天体经过当地子午圈的时刻, 距离天顶最近的时刻称为上中天, 距离天顶最远的时刻称为下中天.

黄道坐标系

  1. 黄道是地球上观察太阳一年中在天球上的视运动所通过的路径. 若以地球不动作为参照的话, 就是太阳绕地球公转的轨道平面在天球上的投影. 这个面叫黄道面
  2. 黄道与天赤道相交于两点, 称为二分点, 分别是春分点和秋分点. 过地球球心垂直于黄道面的直线跟天球的交点叫北黄极和南黄极.
  3. 在天球上与黄道平行的小圆叫黄纬, 垂直黄道过黄极的经度称为黄经, 由春分点起向由西向东度量(0°到350°)
  4. 地轴的进动现象(即岁差), 会导致黄道面变化. 因此坐标数值在使用时需要说明参照系的历元. 现在常采用J2000.0历元.

赤道坐标系

赤道在天球的投影称为天球赤道, 地理极点在天球的投影就是天极.
