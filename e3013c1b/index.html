FLS反向计算

TERMS and Definition

Terms Definition
DDM Difference in Depth of Modulation, \({DDM}_{lat}\) represent DDM in lateral direction or \({DDM}_{ver}\) represent DDM in vertical direction
RT Runway Threshold
AC or A/C Aircraft
AP Anchor Point
RT Altitude The Runway Threshold Altitude is obtained from the following formula:
\({ALT}_{RT} = {ALT}_{AP} - 50 ft\)
Local Level Plane The local level plane for the approach is defined as a plane perpendicular to the local vertical passing through the RT.
FAP Fictitious Anchor Point. It is defined to be the projection of the AP on the local level plane along the local vertical axis.
FLS Beam Course The FLS Beam Course is defined by the following formula: FLS Beam Course = Magnetic Course + Local Magnetic Variation
FFPLO FLS Flight Path Lateral Origin. The FFPLO is defined to be the point situated at 3300 m after the FAP on the local level plane and in the direction of the FLS Beam Course.
Vertical Plane The vertical plane (or lateral deviation reference plane) is defined as a plane containing the FAP and the AP and a line whose direction corresponds to the FLS Beam Course.
Final Approach Path The final approach path is defined as a line in the vertical plane passing through the AP with an angle (defined by Glide Path Angle (GPA) or Slope) relative to the local level plane.
FFPVO FLS Flight Path Vertical Origin. The FFPVO is defined as the intersection of the final approach path and the local level plane (see the following figure).
GPA Glide Path Angle. The GPA is defined as the absolute value of the Slope, i.e. the Glide Path Angle should always be positive whereas the Slope is transmitted as a negative parameter.

Calcuate AC position from DDM

INPUT

  • \({LAT}_{AP}\): AP Latitude
  • \({LONG}_{AP}\): AP Longitude
  • \({ALT}_{AP}\): AP Altitude
  • Slope
  • \(MC\): Magnetic Course
  • \(MV\): Local Magnetic Variation
  • \({DDM}_{lat}\): Lateral DDM
  • \({DDM}_{ver}\): Vertical DDM
  • \(L_{AC-FAP}\): Distance from A/C to FAP

OUTPUT

  • \({LAT}_{AC}\): A/C Latitude
  • \({LONG}_{AC}\): A/C Longitude
  • \({ALT}_{AC}\): A/C Altitude

Solution

\[\phi = \arctan\left(\frac{CourseWidth}{L_{FAP,FFPLO}}\right)\] where: \(CourseWidth = 106.7m\) and \(L_{FAP,FFPLO} = 3300m\)

\[\alpha_{lat} = \frac{DDM_{lat}\times\phi}{0.155}\]

\[ \vec{r}_{AC} = \begin{bmatrix} x \\ y \\ z \end{bmatrix} \]

\[\begin{equation}\label{eq:1}\tan(\alpha_{lat}) = \frac{\vec{u}_{lat}\cdot(\vec{r}_{AC}-\vec{r}_{FFPLO})}{\left|\vec{u}_{long}\cdot(\vec{r}_{AC}-\vec{r}_{FFPLO})\right|}\end{equation}\]

\[\alpha_{vert} = \frac{DDM_{vert}\times 0.25 \times GPA}{0.175}\]

\[\begin{equation}\label{eq:2}\sin(\alpha_{vert} + GPA) = \frac{L_{AC-Projection}}{L_{AC-FFPVO}}\end{equation}\]

\[\begin{equation}\label{eq:3}|\vec{r}_{FAP} - \vec{r}_{AC}| = l\end{equation}\]
